Pour exécuter notre application localement :

1) Créer la base de données en exécutant le script TP4_Livraison.sql situé dans le répertoire : database\TP4_Livraison.sql

2) Se déplacer dans le répertoire server, ouvrir un terminal et faire npm install pour installer les modules, puis npm start pour démarrer le serveur sur le port 3000

3) Se déplacer dans le répertoire client, ouvrir un terminal et faire npm install pour installer les modules, puis npm start pour démarrer le client sur le port 4200

4) ouvrir un navigateur et naviguer vers localhost:4200 pour accéder à notre application.
